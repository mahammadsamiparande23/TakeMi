# MalkHub

A local e-commerce platform connecting village shopkeepers with nearby customers.  
Project started: May 2025  
Author: Mahammad Sami Parande

## Tech Stack
- *Frontend*: React.js, HTML, CSS
- *Backend*: Node.js, Express.js
- *Database*: MongoDB
- *Authentication*: JWT, Passport.js

## Features
- Vendor registration and product listing.
- Buyer registration and product browsing.
- Secure authentication for both vendors and buyers.
- Order placement and tracking.
- Admin management of users and products.

## Installation

1. Clone the repo:  
   git clone <repo-url>

2. Install backend dependencies:  
   cd server  
   npm install

3. Install frontend dependencies:  
   cd client  
   npm install

4. Run the server and client:  
   npm start (for both server and client)
